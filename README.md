## Versión en Español

**SFFALCON** es una agencia digital especializada en diseñar y construir soluciones tecnológicas que impulsan el crecimiento de negocios locales. Nos enfocamos en crear herramientas que no solo se ven bien, sino que capturan clientes y automatizan procesos para liberar el tiempo de los directivos.

### 🚀 Servicios Principales

*   **Webs y Landing Pages**: Diseño profesional de alta conversión optimizado para campañas de marketing.
*   **Aplicaciones Móviles y Web**: Desarrollo nativo y multiplataforma a medida.
*   **SaaS y Paneles de Gestión**: Plataformas personalizadas para gestionar reservas, clientes y operaciones.
*   **IA Integrada**: Agentes inteligentes y automatizaciones 24/7 para atención al cliente y ejecución de tareas.

### 🛠️ Tecnología

Este proyecto está construido con las últimas tecnologías en desarrollo web:

*   **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
*   **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
*   **Estilos**: [Tailwind CSS v4](https://tailwindcss.com/)
*   **Iconos**: [Lucide React](https://lucide.dev/)
*   **Fuentes**: Geist Sans & Mono / Inter

### 📂 Estructura del Proyecto

```text
sffalcon/
├── app/                # Rutas y páginas (Next.js App Router)
│   ├── servicios/      # Páginas de servicios específicos
│   ├── privacidad/     # Políticas de privacidad
│   └── terminos/       # Términos y condiciones
├── components/         # Componentes React reutilizables (Navbar, Footer, etc.)
├── public/             # Activos estáticos (imágenes, iconos)
└── lib/                # Utilidades y configuración (si aplica)
```

### ⚙️ Instalación y Desarrollo

1.  **Clonar el repositorio**:
    ```bash
    git clone https://github.com/markos900313/sffalcon.git
    cd sffalcon
    ```

2.  **Instalar dependencias**:
    ```bash
    npm install
    ```

3.  **Iniciar servidor de desarrollo**:
    ```bash
    npm run dev
    ```

4.  **Construir para producción**:
    ```bash
    npm run build
    ```
